Wartungsarbeiten

Es werden derzeit Wartungsarbeiten am Server durchgefuehrt, die einen gleichzeitigen Spielbetrieb nicht erlauben.

Diese Seite wird alle 5 Minuten fuer Sie probieren, ob das Spiel wieder erreichbar ist. Sobald dem so ist, werden Sie automatisch genau an den Punkt zurueckgelangen, an dem Sie gerade diese Meldung erhalten haben.
